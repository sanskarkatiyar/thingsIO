Instructions to send a request

Say you have the following schema: 

            {
                "field_1": {
                  "name": "temperature",
                  "type": "numeric"
                },
                "field_2": {
                  "name": "pressure",
                  "type": "numeric"
                },
                "field_3": {
                  "name": "gps",
                  "type": "location"
                }
            }

Then you should make a POST HTTP request to the following endpoint: 

  POST http://thingsIO.URL/store/{{ api_key }}

Your request body will contain the following application/json entity:

            {
              "field_1": 123.4,
              "field_2": 443,
              "field_3": [60, 60]
            }

Your request need not contain all the fields' values. You can selectively send specific fields. Location fields should be a [latitude, longitude] pair.
